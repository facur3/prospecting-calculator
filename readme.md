## Setup

1. Ensure `database.js`, `museum_data.js`, and `equipment_data.js` are in the same directory as `index.html`.
2. Open `index.html` in any modern web browser.

## Usage

- **Toggle Layout:** The interface adapts to your screen size. On PC, inputs are arranged in rows for better visibility.
- **Timers:** Keep track of the Traveling Merchant and Infernal Heart events at the top of the screen.
